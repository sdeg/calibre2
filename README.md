# Calibre2
Mini textpad utilisant le localstorage pour la sauvegarde des textes.

## Contexte
J'avais besoin d'un mini outil permettant de taper du texte rapidement, d'avoir une mini-sauvegarde locale. En outre pour le webmastering, il m'est utile de pouvoir taper du texte en markdown et d'avoir le résultat en HTML facilement. Avec l'option Markdown cochée, le bouton "copier" permet d'avoir directement le code HTMl dans le presse-papier.

## Améliorations à venir
